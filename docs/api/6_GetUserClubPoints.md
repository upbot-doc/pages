# 查询用户俱乐部个人积分

## 接口描述

**Path：** /upbot/api/GetUserClubPoints

**Method：** POST


## 请求参数

### Headers

| 参数名称          | 参数值              | 是否必须 | 示例 | 备注 |
|---------------|------------------|------|----|----|
| Content-Type  | application/json | 是    |    |    |
| Authorization | token            | 是    |    |    |

### Body

| 名称                   | 类型         | 是否必须 | 默认值 | 备注                        | 其他信息                                         |
|----------------------|------------|------|-----|---------------------------|----------------------------------------------|
| openid                  | string    | 必须   |     | 心悦openid                    |                                              |


## 请求示例
```
{
    "openid": "JHhd993hGG8n0dsfn9hhsdhfi"
}
```

## 返回数据

| 名称  | 类型      | 是否必须 | 默认值 | 备注 | 其他信息          |
|-----|---------|------|-----|----|---------------|
| ret | integer | 非必须  |     |    |  |
| msg | string  | 非必须  |     |    |               |
| data | object  | 非必须  |     |   数组 |               |
| ├─ amount | integer  | 非必须  |     |   积分数量 |               |



## 成功响应
```
{
    "ret": 0,
    "msg": "",
    "data": {
        "amount": 12000
    }
}
```

## 错误响应
```
{
	"ret": 1001,
	"msg": "请求参数错误，请稍后再试"
}
```
